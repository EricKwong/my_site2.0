About Me

I'm a Web Developer based in New York. I am a graduate of General Assembly's Web Development Immersive course. I love building on the web and the way it allows me to stretch my reach in creativity and thinking. I love playing video games. I've played Counter-strike competitively a third of my life. Playing and being on my computer more than the the average person has driven my interest in technology. I always had a thing for new technologies but I didn't start off striving to become a web developer until I found General Assembly and got a taste of what I can build. I'm also a music lover, tech ethusiast, huge foodie and a sucker for good UX/UI design.

Projects

Check out some of the work I've done

Featur

Featur is an web application that allows users to find a particular music artist and their collaborators. Users are able to see songs the artists collaborated in and save them to their personally created playlists. Featur uses Spotify's API and it allows users to playback songs from their playlists through Spotify Web plugin. Featur was built using Node.js and Backbone.js.

Check it out!

CSGO Veto

CSGO Veto is an app built with React.js and was made for the Counter-Strike Global Offensive competitive gaming community so that users/event organizers can easily veto maps for tournament play.

Check it out!

Fresh Movies

Fresh Movies is a simple app built with React.js, Node.js and utilizes theMovieDB API to retrieve all the latest movies available. It allows users to easily browse through the latest movies and find a short summary and rating on the movies.

Check it out!

Skillmix

Skillmix is a matching service that matches users with users depending on skills they want to learn from each other. Skillmix was built using Node.js and Backbone.js. It also utilizes Amazon Web Services S3 for storing users profile images.
